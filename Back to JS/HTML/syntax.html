Conditional statements:
'if', 'else if' and 'else'.
These have '()' round brackets after the 'if' and 'else if'
that hold the conditions that need to be met such as
'if(myVariable == true)' or 'else if(myVariable == 6)'.
After the condition, you have curly brackets '{}' that house what needs to be done. Eg:


            if(myVariable == true) {
                console.log("Message is true.");}
            else {
                console.log("Message is not true.");
            }

The 'else if' and 'else' can be used to add more conditions.

Arrays and loops:
An array holds multiple variables for us, such as names or numbers or even our own custom classes.
The first index of an array is zero (0), but is counted as one (1) when we ask how many slots are in the array.
Eg; 'let myArrayOfNumbers = [1, 2, 3, 4, 5, 6, 7, 8, 9];'
So the number '9' would be at index 8, but there are 9 slots in the array. I know, I know...
Loops can be 'for' or 'while'.
'for' loops run FOR each number of times you tell it to,
while 'while' loops run WHILE there is a condition that is true. Eg;


            for (i = 0; i < 10; i++) {
                printing out 'i' will count to ten.
            }
            while(something == true) {
                this will repeat until that something becomes false.
            }

Maps:
A map is like an array from earlier, but the big difference is that every item has an index name,
and then a value attached to that specific index. Eg; 


            let treasureMap = new Map();
            treasureMap.set("index-1", "this is where my first treasure is buried");
            treasureMap.set("index-2", "this is where my second treasure is buried");

Now if we want to know where our treasure is, we go to our Map and call for
the index with either 'index-1' or 'index-2' to get our respective values;



            let myNewMap = new Map();
            myNewMap.set("the quiver", "where green arrow has his hideout.");
